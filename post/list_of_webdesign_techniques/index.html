Web系技術のまとめ一覧

category: 開発環境

  • #Web制作

Jul 24, 2015

Webに関する技術を俯瞰的に見るための一覧。個人的で偏りあり

    • 全般

全般

≒ は同等・類似機能を持つもの

  • 上流 (設計、フロー etc.)
  • Webデザイン、UI/UX
  • SEO/SEM
  • Webライティング
  • セキュリティ
  • プログラミング基本 (アルゴリズム etc.)
  • 開発ツール、開発手法
    • Vagrant + VirturalBox
    • CoreOS + Docker
    • Yeoman, gulp.js, Bower JavaScript>ビルドツールの項を参照
    • Sublime Text
  • HTML
    • HTML/CSSコーディング、レスポンシブデザイン
    • HTML5（Drag&Drop API,Web Workers,Canvas,Web Storage,WebSocket 他）
    • Jade ≒ ejs,haml,Slim
  • CSS
    • CSS3
    • Sass(SCSS) ≒ LESS,Stylus
    • StyleDocco スタイルガイド自動作成、gulpで自動化
  • JavaScript
    • Pure JavaScript
    • jQuery ライブラリ（DOM操作、クロスブラウザ対応 他） ≒ zept(jQuery互換)
    • Underscore.js JavaScriptユーティリティ・ライブラリ ≒ Lo-Dash(Underscore.js互換版/拡張版)
    • Handlehars JSテンプレートエンジン ≒ Mustache,Hogan.js(Mustache拡張),Underscore.jsの_.templateメソッド
    • D3.js データ可視化ライブラリ
    • node.js
      • CoffeeScript JavaScriptがより短く簡素に書ける言語。JavaScriptにコンパイルして使用
      • mocha node上で動くJavaScriptテストフレームワーク
      • MEAN Stack
        • MongoDB →データベースの項を参照
        • express node.jsのMVCフレームワーク
        • Angular.js フロントエンド用MVWフレームワーク。HTMLがテンプレート ≒ backbone.js,Knockout.js,ember.js
        • node.js(nodebrew：バージョン管理,npm：パッケージ管理,forever：永続化,async.js：フロー制御)
      • ビルドツール、タスクランナー、自動化、パッケージ管理
        • Yeoman(yo) 雛形作成ツール。下記2つをセット使用。Webアプリに比重 ≒ Middleman(Ruby製、静的サイト向け)
        • gulp.js タスクランナー ≒ Grunt,Brunch,Codekit
        • Bower フロントエンド(js,css,html)のパッケージマネージャ ≒ Jam
    • React.js(Facebook製UIライブラリ、中規模以上向け、jQuery併用難しい) »解説 »解説2 » 解説3 解説4
    • Browserify JSモジュール管理。Debowerifyプラグイン併用でnpmだけでなくbower管理のライブラリも扱える ≒ require.js (»解説)
    • テスト用
      • jasmine 2 JSテストフレームワーク ≒ mocha + chai(アサーションライブラリ) + sinon.js(モックライブラリ)
      • PhantomJS（ヘッドレスブラウザ）
      • karma(リモートテストランナー）
      • travis.yml（継続的インテグレーション）
    • tmlib.js ≒ enchant.js ゲーム開発
    • Titanium mobile モバイル向けクロスプラットフォーム開発ツール ≒ cordova(PhoneGap),Sencha Touch
    • WebComponents HTMLコンポーネント化(パーツの再利用容易に)。Web標準になる?、Google製 (»解説) 解説2 »解説3 (»解説4)
      • Polymer WebComponentsのラッパーライブラリ。polyfill、データバインディング、アニメーション他含む。Google製 (»解説)
  • PHP
    • ライブラリ、フレームワーク
  • CMS、静的サイトジェネレーター
    • WordPress
    • Hugo
  • その他サーバーサイドの言語
  • データベース
    • MongoDB NoSQLドキュメント指向DB（JSと親和性が高くnode.js+Mongooseとの組み合わせでの使用が人気）
    • MySQL/MariaDB オープンソースのRDBMS ≒ PostgreSQL
    • SQLite 軽量RDBMS。Mac,iPhone,Androidに標準搭載
    • lawnchair LocalStorageへ簡単にCRUDできるシンプルなJSONストレージ
    • FileMaker
  • Web API
    • OAuth 2.0 (クライアントサイド/サーバーサイド)
  • iOSアプリ開発
    • Swift ≒ Objective-C
    • XCode
    • Cocos2D-Swift (クロスプラットフォーム ゲームフレームワーク)
  • AppleScript
  • バージョン管理
    • Git、Github、Bitbucket、GitBucket
  • サーバー
    • コマンド(bash)
    • Vagrant 仮想サーバー管理
    • VirtualBox 仮想化ソフト
    • 構成管理ツール Ansible(設定はYAML), Chef, Puppet
    • Heroku
    • VPS(さくらVPS, VULTR, Digital Ocean)
    • Docker コンテナ型仮想化。サーバー環境をエミュレート。Gitライクなバージョン管理も可能（わかりやすい説明、Mac上で動かす方法）
      • Fig マルチコンテナ構成ツール。複数のコンテナを1つのシステムとして定義するための正式な方法となりそう Docker社がマルチコンテナ構成ツール「Fig」を採用。開発した社員2名のOrchardを買収 － Publickey
    • AWS Amazon
    • Amazon Cognito モバイルデバイスでローカルのデータストアと認証機能。クラウド上のデータストアとの同期機能（わかりやすい説明）
    • Amazon Zocalo Dropboxライクなクラウド・ストレージ。Active Directory(Windows Server)と連携可能なするユーザー管理機能を備える。月額5ドルで200GB（わかりやすい説明）
    • Azure
    • Heroku
    • Apache, Nginx
